\section{電子天秤}
各材料の計量には電子天秤(島津製作所製、AEX-120G)を用いた。以下に測定手順を示す。
\\\newline
測定の準備\\
　天秤を使用しない時でも電源プラグは抜かずに、ON/OFFキーを押してスタンバイランプを点灯させ、暖機状態に保つ。また、計画停電などで電源を切った後再起動する際には、暖機を一時間半以上行う。再起動時は、電子天秤の表示が\SIrange[range-phrase = { or }]{0.0000}{0.000}{ \g}のどちらかになるため注意する必要がある。
\\\newline
測定モードの設定
\begin{enumerate}
    \item ON/OFF キーを押す。スタンバイランプが消え、全オブジェクトが点灯する。点灯しないオブジェクトがないかどうか確かめる。
    \item TARE キーを押し、感度校正を行う。校正結果が表示され、''CAL End''が表示されると校正が終了する。
    \item 表示がゼロになり、測定モードになる。
\end{enumerate}
\leavevmode\\
測定
\begin{enumerate}
    \item 薬包紙を使用するときは皿に薬包紙を載せ、安定マーク（→）が点灯後 TARE キーを押し、零点補正を行う。
    この時、電子天秤の全ての窓は閉めておく。
    \item 表示がゼロであることを確認する。
    \item 窓を開け、材料を薬包紙の上に載せ、窓を再度閉める。
    \item 安定マークが点灯後、表示を読み取る。
    \item 測定終了後、ON/OFF キーを押しスタンバイ状態にする。
 \end{enumerate}   
    
\section{試料の作製}
本研究では、基板上にspiroOMeTADまたは中間層を成膜した。以下にその詳細を示す。
\subsection{基板の洗浄}
ITO付きガラス基板は、表面の汚れによる電気特性や堆積する物質の結晶構造への影響を防止するため、使用前に洗浄工程を行い保管する。基板の洗浄には、超音波洗浄機（本多電子株式会社：W-113）を用いた。周波数は\SIlist{100}{\kilo\hertz}に設定した。以下にその手順を示す。

\noindent\textbf{洗浄工程}
\begin{enumerate}
    \item 基板洗浄用の\SI{500}{\milli\liter}ビーカーに脱イオン水と基板を入れ、超音波洗浄を\SI{100}{\kilo\hertz}で10分間行う。この過程を6回繰り返す。
    \item 脱イオン水を捨て、2-プロパノールを入れて超音波洗浄を\SI{100}{\kilo\hertz}で15分間行う。
    \item 2-プロパノールを廃液入れに捨て、基板をソックスレー抽出器に移す。
    \item ソックスレー抽出器に冷却水を流す。
    \item 電源プラグを接続し、スライダックの目盛りを7に合わせ、2-プロパノールの還流を3サイクル行う。
    \item 還流終了後、スライダックの電源を切り、基板はそのまま2-プロパノールに浸した状態で保管する。
    \item ソックスレー抽出器が十分に冷えたことを確認し、冷却水を止める。
\end{enumerate}

	\end{enumerate}

		\item 共洗いに使用した溶媒を廃液入れに廃棄する。
		\item 溶媒をふき取った後、所定の位置に戻す。
	\end{enumerate}

対極には白金電極\SIlist{5.7}{\cm}(ビー・エー・エス株式会社製、002222)を使用した。使用後、対極は脱イオン水で十分にすすぎ、常温で乾燥させる。\\
\quad  参照極\\
参照極には銀-塩化銀電極(ビー・エー・エス株式会社製、RE-1B)を使用した。以下この電極をAg/AgClと表記する。
\subsection{測定手順}
セル準備
\begin{enumerate}
    \item 電解質を電子天秤で計量し、必要量を電気化学セル(ビー・エー・エス株式会社製、SVC-2)に入れる。
    \item 溶媒を必要量、メスシリンダーで測り、電解質を入れた電気化学セルに入れる。
    \item 電気化学セルにパラフィルムで蓋をして、電解質の濃度が均等になるように振り混ぜる。
    \item パラフィルムを取り外す。
    \item SVC-2 用テフロンキャップの穴から作用極に取り付ける導線を通し、みのむしクリップを作用極に接続する。
    \item 作用極を電解質溶液に浸漬させ、セルキャップを電気化学セルに取り付ける。
    \item 参照極の底の気泡を、指で弾いて取り除いた後、参照極を電解質溶液に浸漬させる。
    \item 対極を電解質溶液に浸漬させる。
    \item それぞれの電極に HSV-110 のセルコードを接続する。
\end{enumerate}
　装置準備
\begin{enumerate}
    \item HSV-110 がコンセントに接続されている事、HSV-110 と PC が LAN ケーブルで接続されている事を確認し、HSV-110 の電源を入れる。
    \item HSV-110 が起動した事を確認し、操作部にある STORE キーと LOCK キーとを操作し、表示ランプ (緑色) を点灯させ、PC による操作準備を完了させる。
    \item PC 上で、HSV-110REMOTE を起動する。
    \item 起動画面上のMEASURE をクリックし、リモート制御画面を起動させる。
\end{enumerate}
　測定
\begin{enumerate}
    \item リモート制御画面上で測定方法、測定パラメータを設定する。
    \item START/STOP ボタンをクリックし、測定を開始する。
    \item 測定終了後、測定データ保存ダイアログが開くため任意のディレクトリに保存を行う。
\end{enumerate}
　片付け
\begin{enumerate}
    \item 作用極、対極、参照極に接続されたセルコードを外す。
    \item 参照極は脱イオン水で濯いだ後、\SIlist{3}{\mole/\liter} \ce{NaCl} aq. に液絡部を浸して保管する。
    \item 対極は脱イオン水で濯いだ後、水滴を拭き取り、所定の場所に保管する。
    \item 作用極を電気化学セルから取り出し、使用した電解質溶液は廃液入れに捨てる。
    \item 電気化学セルを使用した溶媒で数回濯いだ後、自然乾燥させる。
    \item END ボタンをクリックし、リモート制御画面を閉じる。
    \item 起動画面の終了ボタンをクリックする。
    \item 測定停止状態である事を確認して、装置の電源スイッチをオフにする。
\end{enumerate}

サイクリックボルタンメトリーをする際に、電解液の濃度が低いと酸化に関係のない電流が多く流れてしまいその結果、還元電流が小さくなってしまうと言う現象が確認されている。これは、高濃度電荷液を用いる事によって解決できるということが先行研究で分かっている。\cite{山本雄樹}図\ref{高濃度電解液}に電解液の濃度を変えていったときのサイクリックボルタモグラムを示す。

\begin{figure}[H]
	\centering
	\includegraphics[width=10cm]{image/High Electrolyte.png}
	\caption{溶液濃度別のサイクリックボルタモグラム\cite{山本雄樹}}
	\label{高濃度電解液}
\end{figure}

図\ref{高濃度電解液}より、電解液の溶液濃度が濃くなるほど$\SI{1.0}{V}$\,vs.Ag/AgCl以降に流れる電流密度が小さくなっているのがわかる。これは、酸化に関係のない電流が流れにくくなったことを示している。このことから、本研究では電解液の溶液濃度を$\SI{10}{mol/kg}$にして測定を行った。
